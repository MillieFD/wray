## Project Objectives

The `wray` project provides researchers with a standardised optical spectroscopy data format.

1. **Performant:** High-throughput reading and writing
2. **Efficient:** Suitable for edge deployment on resource-constrained hardware
3. **Compact:** Keep file sizes small for easy data sharing
4. **Ergonomic:** Users see a single `.wray` file on disk which can be easily moved & shared
5. **Flexible:** Can be adapted to a wide variety of experiments
